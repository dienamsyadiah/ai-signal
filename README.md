# AI Signal - Crypto Market Analysis

AI Signal adalah aplikasi berbasis web yang digunakan untuk membantu menganalisis pergerakan market crypto berdasarkan timeframe tertentu. Sistem ini dirancang untuk memberikan insight apakah pergerakan candlestick berpotensi naik atau turun berdasarkan hasil analisis data market.

Project ini dibuat sebagai sistem pendukung analisis, khususnya untuk membantu pengguna membaca arah pergerakan market crypto secara lebih terstruktur.

## Fitur Utama

* Analisis pergerakan candlestick crypto
* Prediksi arah market berdasarkan timeframe tertentu
* Insight potensi candlestick bergerak naik atau turun
* Dashboard analisis market
* Tampilan aplikasi berbasis web
* Dokumentasi hasil analisis
* Sistem pendukung keputusan untuk analisis crypto

## Akurasi Analisis

Berdasarkan hasil pengujian internal/backtesting pada data yang digunakan, sistem ini menunjukkan tingkat ketepatan analisis sekitar **88%** dalam membaca potensi arah pergerakan candlestick.

> Catatan: Hasil analisis tidak dapat dijadikan jaminan keuntungan. Market crypto memiliki risiko tinggi dan dapat berubah secara cepat. Sistem ini digunakan sebagai alat bantu analisis, bukan sebagai keputusan investasi mutlak.

## Cara Menjalankan Project

1. Clone repository ini.

```bash
git clone https://github.com/dienamsyadiah/ai-signal.git
```

2. Masuk ke folder project.

```bash
cd ai-signal
```

3. Install dependency.

```bash
npm install
```

4. Jalankan project.

```bash
npm run dev
```

5. Buka aplikasi di browser.

```txt
http://localhost:3000
```
